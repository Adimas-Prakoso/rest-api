Extract structured data from any website with our powerful scraping API

POST /scraping

Request Parameters

url
URL of the website to scrape (required)

Example Request

{
    "url": "https://example.com"
}

Example Response

{
    "title": "Example Website",
    "description": "Website description from meta tags",
    "links": [
        {
            "url": "https://example.com/page1",
            "text": "Link Text"
        }
    ],
    "images": [
        {
            "url": "https://example.com/image.jpg",
            "alt": "Image description"
        }
    ],
    "text": [
        "Content from paragraph 1",
        "Content from paragraph 2"
    ]
}

Response Codes

200 Success
Data successfully scraped from website
400 Bad Request
Invalid URL or URL not provided
500 Server Error
Failed to scrape website data

Features

Structured Data

Extracts data in organized JSON format including title, description, links, and images

Meta Information

Automatically captures meta tags and SEO-related information

Content Extraction

Pulls text content from paragraphs and other relevant elements

Important Notes

  • URL must be valid and accessible
  • Some websites may block scraping attempts
  • Response time may vary based on website size and complexity
  • Consider implementing rate limiting in your application
  • Respect robots.txt and website terms of service
